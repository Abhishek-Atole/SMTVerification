🆘 QUICK TROUBLESHOOTING:

  Problem: "No columns found"
  Fix: Use standard column names (see examples)

  Problem: "No valid data"
  Fix: Ensure Item Name + Reference not empty

  Problem: File won't load
  Fix: Use .csv, .xlsx, or .xls format

  Problem: Numbers showing as 0
  Fix: Ensure Qty/SR are numeric, not text

  Problem: Encoding issues
  Fix: Save CSV as UTF-8, not ANSI
